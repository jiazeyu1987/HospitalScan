# 全国医院监控系统 - 前端数据查看指南

## 📊 数据源说明

### ✅ 前端数据配置
- **API地址**: `http://localhost:5000/api/v1`
- **数据源**: 真实后端API（无模拟数据）
- **连接方式**: Axios HTTP请求
- **响应格式**: JSON格式

### 🗄️ 后端数据状况
- **数据库**: SQLite (hospital_monitor_dev.db)
- **行政区划**: 483个记录（4级树形结构）
- **系统设置**: 10个配置项
- **定时任务**: 4个已启动（监控、扫描、报告）

## 🔍 实际操作演示

### 步骤1: 启动后端服务
```bash
cd backend
python run.py
```
访问: http://localhost:5000

### 步骤2: 启动前端服务
```bash
cd hospital-monitor-antd
npm run dev
```
访问: http://localhost:3000

### 步骤3: 查看医院信息
1. **主页面**: http://localhost:3000/
   - 查看系统统计概览
   - 快速操作入口

3. **招投标监控**: http://localhost:3000/tenders
   - 查看招投标数据
   - 按医院筛选
   - 导出Excel

4. **爬虫控制台**: http://localhost:3000/crawler
   - 启动/停止扫描
   - 查看实时状态
   - 监控扫描进度

5. **系统设置**: http://localhost:3000/settings
   - 配置扫描参数
   - 设置通知选项

## 🔗 API接口参考

### 医院相关接口
```
GET /api/v1/hospitals          # 获取医院列表
GET /api/v1/hospitals/{id}     # 获取医院详情
POST /api/v1/hospitals         # 创建医院
PUT /api/v1/hospitals/{id}     # 更新医院
DELETE /api/v1/hospitals/{id}  # 删除医院
GET /api/v1/hospitals/search   # 搜索医院
POST /api/v1/hospitals/batch_import  # 批量导入
```

### 招投标相关接口
```
GET /api/v1/tenders            # 获取招投标列表
GET /api/v1/tenders/{id}       # 获取招投标详情
POST /api/v1/tenders/export    # 导出Excel
GET /api/v1/tenders/stats      # 获取统计数据
```

### 地区相关接口
```
GET /api/v1/regions            # 获取地区树形结构
GET /api/v1/regions/{id}/hospitals  # 获取地区下医院列表
```

### 爬虫相关接口
```
POST /api/v1/crawler/start     # 启动爬虫
POST /api/v1/crawler/stop      # 停止爬虫
GET /api/v1/crawler/status     # 获取爬虫状态
GET /api/v1/crawler/logs       # 获取爬虫日志
```
